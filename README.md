How to Disable “Let’s Finish Setting up Your Device” on Windows 11

To turn off “let’s finish setting your device” on Windows 11, you can go to Settings to do that:

Step 1: Open the Settings app by pressing the Windows + I keys together.

Step 2: Select the System option and click Notifications on the sidebar.

Step 3: Uncheck the Offer suggestions on how I can set up my device option.
